# Парсер индексов Почты России

Небольшой PHP-скрипт, который скачивает актуальный архив индексов с сайта Почты России, достает из него DBF-файл и собирает CSV `pochta_indexes.csv`.

В CSV попадают индекс, регион, район, город, населенный пункт, название и тип отделения, дата актуальности, старый индекс и примерная таймзона региона.

## Запуск

```bash
php parse_pochta.php
```

После запуска скрипт:

1. скачает страницу справочника Почты России;
2. найдет ссылку на ZIP-архив;
3. сохранит архив и DBF во временную папку `pochta_tmp/`;
4. перезапишет файл `pochta_indexes.csv`.

Если все прошло успешно, в консоли появится путь к CSV, количество строк и даты обновления архива/DBF.
